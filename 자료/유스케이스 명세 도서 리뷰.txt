유스케이스 명세: 도서 리뷰


개요

이용자는 보고자 하는 도서의 리뷰 페이지로 이동한다. 리뷰 페이지에는 리뷰 조회, 리뷰 작성, 리뷰 수정, 리뷰 삭제 기능이 표시된다.
해당 도서를 구매한 상태가 아닌 경우 구매 페이지로 이동한다.

관련 액터
주액터: 서비스 이용자
보조액터: 없음
우선순위
중요도: 상, 난이도: 상

선행 조건
이용자는 시스템에 로그인을 한 상태이어야 한다.
이용자는 해당 도서를 구매한 상태이여야 한다.

후행 조건
작성,수정,등록,삭제된 리뷰는 데이터베이스에 반영된다.

시나리오
기본 시나리오:

1. 이용자는 홈 화면에서 “리뷰” 버튼을 선택한다.
2. 시스템은 리뷰 화면을 출력한다.
3. 이용자는 “뒤로” 버튼을 선택한다.
4. 시스템은 홈 화면을 출력한다.

대안 시나리오:

A1 리뷰작성: 기본 시나리오 2에서 분기
1. 이용자는 리뷰 화면에서 리뷰 등록을 선택한다.
2. 시스템은 현재 이용자가 소장중인 도서 목록을 출력한다.
3. 이용자는 리뷰하고자 하는 도서를 선택한다.
4. 시스템은 해당 도서의 페이지 목록을 출력한다.
5. 이용자는 리뷰를 작성할 위치를 선택한다.
6. 시스템은 리뷰 작성 화면을 출력한다.
7. 이용자는 리뷰를 작성한다.
8. 이용자는 “저장” 버튼을 선택한다.
9. 시스템은 리뷰 화면을 출력한다.

A2 리뷰읽기: 기본 시나리오 2에서 분기
1. 이용자는 리뷰 화면에서 리뷰 읽기를 선택한다.
2. 시스템은 현재 이용자가 소장중인 도서 목록을 출력한다.
3. 이용자는 리뷰를 읽고자 하는 도서를 선택한다.
4. 시스템은 해당 도서의 페이지 목록을 출력한다.
5. 이용자는 리뷰를 읽을 위치를 선택한다.
6. 시스템은 해당 위치의 리뷰 목록을 출력한다.
7. 사용자는 읽고자 하는 리뷰를 선택한다.
8. 시스템은 선택된 리뷰 화면을 출력한다.
9. 사용자가 “뒤로” 버튼을 선택한다.
10. 시스템이 리뷰화면을 출력한다.

A3 리뷰수정: 기본 시나리오 2에서 분기
1. 이용자는 리뷰 화면에서 리뷰 수정을 선택한다.
2. 시스템은 이용자가 작성한 리뷰의 리스트를 출력한다.
3. 이용자는 수정하고자 하는 리뷰를 선택한다.
4. 시스템은 해당 리뷰의 수정 화면을 출력한다.
5. 이용자는 리뷰를 수정한다.
6. 이용자는 “리뷰 저장” 버튼을 선택한다.
7. 시스템은 리뷰화면을 출력한다.

A4 리뷰삭제: 시나리오 2에서 분기
1. 이용자는 리뷰 화면에서 리뷰 삭제를 선택한다.
2. 시스템은 이용자가 작성한 리뷰의 리스트를 출력한다.
3. 이용자는 삭제하고자 하는 리뷰를 선택한다.
4. 시스템은 리뷰화면을 출력한다.

비기능적 요구사항
해당 없음
